# Example Plugin

A demonstration plugin showing all Claude Code plugin component types.

## Components

- **Commands**: Slash commands for user-invoked actions
- **Agents**: Autonomous specialist agents
- **Skills**: Context-aware guidance
- **Hooks**: Event-driven automation

## Installation

```bash
/plugin marketplace add /path/to/claude-plugin-boilerplate
/plugin install example-plugin@claude-plugin-boilerplate
```

## Usage

This plugin demonstrates:
- Creating slash commands with argument handling
- Building autonomous agents with specific tools
- Writing skills with clear trigger conditions
- Implementing hooks for event automation

## Development

Use this plugin as a reference when creating your own plugins.
