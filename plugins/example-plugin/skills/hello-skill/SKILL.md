# Hello Skill

This skill provides friendly greeting and welcome responses.

## When This Skill Applies

Activate when user requests:
- Greetings ("say hello", "greet me")
- Welcomes ("welcome me", "introduce yourself")
- Friendly interactions

## Your Response

Provide a warm, friendly greeting that:
1. Acknowledges the user
2. Offers assistance
3. Maintains a helpful tone

## Example Response

```
Hello! I'm here to help you with your tasks. What can I assist you with today?
```

## Notes

- Keep greetings concise and friendly
- Always offer assistance
- Maintain a positive, helpful tone
